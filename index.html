Engineering Calculations & Standards

This sizing and selection engine performs code compliance checks using standard international regulations:

  • ASME B31.3 Process Piping: The pressure design wall thickness is calculated using the straight pipe formula:
    t = (P * D) / (2 * (S*E + P*Y))
    Where:
    - P = Design gauge pressure
    - D = Outside diameter (ASTM B36.10/19)
    - S = Material allowable stress at temperature (ASME B31.3 Table A-1)
    - E = Quality factor (1.0 for seamless pipe)
    - Y = Material coefficient (0.4 for steels)
  • Manufacturing Mill Tolerance: Standard steel pipe is manufactured to a -12.5% wall thickness tolerance. The nominal design thickness includes this factor:
    t_nom = (t_design + corrosion) / 0.875
  • Chemical Compatibility: Compatibility data is cross-referenced dynamically from industrial compatibility database of metallic components and elastic seals.
